OCR PDF Extractor for Valuation Reports

An automation pipeline that extracts key numeric fields from valuation PDFs using OCR, then aggregates them into an exception report product for internal stakeholders. Built to reduce manual review time and improve accuracy for Zweig Group.

Key Capabilities:

  • OCR Pipeline: Tesseract-based OCR with preprocessing for noisy scans.
  • Targeted Extraction: Regex and layout-aware parsing for metrics (fees, dates, amounts).
  • Validation: Confidence thresholds and rule checks to flag anomalies.
  • Export: CSV/Excel output for the exception report product.
